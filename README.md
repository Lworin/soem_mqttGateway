# soem_mqttGateway

Projeto final da disciplina de Sistemas Operacionais Embarcados

Arquivos:

* `main.py` código principal
* `DBConnection.py` classe para gerenciar a conexão com o banco no banco de dados
* `Message.py` classe para gerenciar a inserção dos pacotes no banco de dados
* `SetupUbuntu.md` arquivo com passo-a-passo de configurações feitas no Ubuntu
